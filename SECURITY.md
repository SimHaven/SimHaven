# Security Policy

## Supported versions

Only the latest publicly released version of SimHaven is actively supported. Older versions may not receive security fixes.

## Reporting a vulnerability

Do not report a suspected vulnerability through a public GitHub issue, discussion, pull request, or community chat.

Use GitHub's private vulnerability-reporting feature when it is available for this repository. Alternatively, email [wyatt@simhavenserver.com](mailto:wyatt@simhavenserver.com).

Include:

- A description of the issue and its potential impact
- The affected version and component
- Reproduction steps or a minimal proof of concept
- Any suggested mitigation

Do not include credentials, player data, database contents, or information taken from SimHaven's live systems. If sensitive evidence is necessary, first ask how it should be transferred.

SimHaven will review reports as availability permits. We cannot promise a specific response or resolution time. Please allow a reasonable opportunity to investigate before public disclosure.

## Security updates

Confirmed fixes will be distributed through the normal SimHaven release or update process when they are ready and have been tested.
